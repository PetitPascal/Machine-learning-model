This folder contains the trained machine learning model generated for the study titled “Telemedicine adoption in cardiology: determinants and predictors identified using Bayesian model averaging and machine learning”, made available to support reproducibility and external validation.

If you use any model from this repository, please cite the corresponding study:
Petit P, Nübel J, Walter J, Butter C, Heinze M, Ignatyev J, Haase-Fielitz A, Vuillerme N, Muehlensiepen F. Telemedicine adoption in cardiology: determinants and predictors identified using Bayesian model averaging and machine learning. PLOS Digital Health. 2026. Accepted
